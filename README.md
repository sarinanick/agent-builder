# 🤖 Agent Builder

A visual AI Agent workflow builder, inspired by OpenAI Agent Builder. Build complex multi-agent workflows with a drag-and-drop canvas editor.

## ✨ Features

- **Visual Canvas Editor** - Drag & drop nodes, zoom, pan, minimap, animated edges
- **12 Custom Node Types** - Start, End, Agent, Note, File Search, Guardrails, MCP, Condition, While, Human Approval, Transform, Set State
- **Properties Panel** - Configure each node with dedicated forms
- **Dark/Light Theme** - Toggle between themes
- **Undo/Redo** - Full history support
- **Code Export** - ChatKit + Agents SDK
- **Preview Mode** - Test workflows with simulated chat
- **Deploy Options** - Vercel, self-hosted, download
- **Sample Template** - Pre-loaded "Homework Helper" workflow

## 🚀 Getting Started

```bash
# Clone
git clone https://github.com/your-username/agent-builder.git
cd agent-builder

# Install
npm install

# Dev
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)
